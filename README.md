💻 Sistema Bancário em Java

Este projeto foi desenvolvido como parte da minha jornada de aprendizado em Java, com foco em aplicar na prática conceitos fundamentais de programação orientada a objetos e organização de código.

A aplicação simula operações básicas de um banco, como:
- Criação de contas
- Depósitos
- Saques
- Cálculo do saldo total do banco

Durante o desenvolvimento, trabalhei conceitos importantes como:
- Encapsulamento
- Separação de responsabilidades (model, service, app)
- Boas práticas de nomenclatura
- Manipulação de entrada de dados com Scanner
- Estruturação de um projeto Java de forma mais organizada

Também busquei melhorar a qualidade do código, removendo acoplamentos desnecessários e deixando as regras de negócio mais bem definidas.

OBS: Esse projeto representa minha evolução prática no aprendizado de Java e marca um passo importante na construção de uma base sólida para projetos maiores.
